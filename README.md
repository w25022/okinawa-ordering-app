# 沖縄ハイボール酒場 三線 (Okinawa Ordering App)

本プロジェクトは、沖縄料理を提供する居酒屋のメニュー閲覧および注文システムを模したWebアプリケーションです。直感的なUIレイアウト、多言語切り替え機能、およびカートシステムを搭載しています。

## 主な機能

- **カート管理機能**
  - メニューカード上の追加ボタンより、リアルタイムで数量の増減が可能です。
  - カート画面にて選択された商品の詳細確認、削除、および合計金額の自動計算を行います。

- **多言語対応機能**
  - アプリ起動時の初期画面およびヘッダーから、日本語、英語、繁体字中国語、簡体字中国語、韓国語、スペイン語の6言語に表示を切り替えることができます。

- **初期確認システム**
  - アプリ起動時に利用人数の確認、および20歳未満へのアルコール提供禁止に関する規約同意を求める画面を表示する設計となっています。

- **カテゴリフィルター**
  - メニューを「すべて」「おすすめ」「とりあえず」「お肉料理」「炒めもん」「ご飯・麺・ピザ」「ドリンク」のカテゴリに分類し、横スクロールバーで迅速に切り替えられます。

- **注文履歴の保存**
  - 確定された注文データはブラウザの localStorage に蓄積され、履歴画面からいつでも過去の注文内容を確認することができます。

## ファイル構成

- index.html   : アプリケーションの画面構造の定義
- style.css    : モバイル表示に最適化したレイアウトおよびスタイリング
- script.js    : 多言語処理、カート計算、データ保存等のロジック処理
- images/      : メニューに使用する画像資産

## 技術スタック

- HTML5
- CSS3 (Variables、Flexbox、Gridの活用)
- JavaScript (ES6+、DOM操作、localStorage API)

## 使用方法

本アプリケーションはサーバー環境の構築を必要とせず、ブラウザ環境のみで動作します。

1. 本リポジトリのファイルをローカル環境にダウンロードします。
2. フォルダ内の index.html をダブルクリックし、任意のブラウザで起動します。
3. モバイル端末での表示に最適化されているため、ブラウザのデベロッパーツール等でスマートフォンサイズに縮小して閲覧することを推奨します。
