# Okinawa Ordering App

このプロジェクトは、沖縄料理をテーマにしたシンプルな注文アプリです。

## 現在の状態

- 多言語対応（日本語、英語、繁体字中国語、簡体字中国語、韓国語、スペイン語）
- カテゴリごとのメニュー表示
- カートに追加、数量変更、削除
- カート合計金額表示
- 注文ボタンで注文完了メッセージを表示

## 目的

このアプリをGitHubで公開し、今後も継続して開発していきます。UIの改善、注文機能の強化、バックエンド連携などを進める予定です。

## ファイル構成

- `index.html` - ページ構成
- `style.css` - デザイン
- `script.js` - アプリのロジック
- `images/` - メニュー画像
